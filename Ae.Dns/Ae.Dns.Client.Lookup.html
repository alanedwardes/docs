Namespace Ae.Dns.Client.Lookup

Classes

DhcpdConfigDnsLookupSource

Provides a IDnsLookupSource which parses DHCPD config files, reading the IP address and hostname, and adding it to the lookup table for DNS resolution.

DhcpdLeasesDnsLookupSource

Provides a IDnsLookupSource which parses DHCPD leases files, reading the IP address and hostname, and adding it to the lookup table for DNS resolution.

DictionaryDnsLookupSource

Provides a IDnsLookupSource implementation based on dictionaries.

DnsStaticLookupClient

A DNS client providing a static lookup (similar to a hosts file). If the host is not found in that lookup, the request is forwarded to the other specified IDnsClient.

FileDnsLookupSource

An abstract implementation of IDnsLookupSource which watches the specified file for changes, and calls ReloadFile().

HostsFileDnsLookupSource

Provides a IDnsLookupSource which monitors the specified hosts file.

Interfaces

IDnsLookupSource

Describes a service capable of resolving hostnames to IP addreses, and IP addresses to hostnames.
